# iconfont-mcp
iconfont 图标助手, 基于 mcp 协议的图标下载助手

## 安装
```bash
npx iconfont-mcp
```
## MCP 使用
```json
{
  "mcpServers": {
    "iconfont": {
      "command": "npx",
      "args": [
        "iconfont-mcp"
      ]
    }
  }
}
```

## 开发
```bash
npm run dev
```

## 构建
```bash
npm run build
```


## 调试
```bash
npx @modelcontextprotocol/inspector

```
